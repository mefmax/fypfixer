# СРОЧНО: Нужны TikTok Credentials
# Дата: 2025-12-27

---

## ПРОБЛЕМА

Frontend не может авторизовать через TikTok OAuth.

URL показывает: `client_key=undefined`

Это значит что `VITE_TIKTOK_CLIENT_KEY` не задан в Frontend/.env

---

## ЧТО НУЖНО

Добавь в `Frontend/.env` следующие переменные:

```env
VITE_TIKTOK_CLIENT_KEY=<твой_client_key>
VITE_TIKTOK_REDIRECT_URI_DEV=http://localhost:5173/auth/tiktok/callback
VITE_TIKTOK_REDIRECT_URI_PROD=https://fypglow.com/auth/tiktok/callback
```

---

## ВОПРОС

Где взять VITE_TIKTOK_CLIENT_KEY?

1. TikTok Developer Portal: https://developers.tiktok.com/
2. Твое приложение → App Info → Client Key

---

## АЛЬТЕРНАТИВА

Если TikTok OAuth не нужен для Day 6 разработки компонентов,
можно просто не заходить на страницу логина и работать с UI.

Подтверди какой вариант выбираем:
A) Дай мне Client Key — я добавлю в .env
B) Игнорируем OAuth — работаем только над UI компонентами
